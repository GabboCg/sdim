**Date:** 2026-04-04
**Status:** Approved

## Overview

Add six lazy-load datasets to the `sdim` package sourced from the replication package of He, Huang, Li, Zhou (2023) doi:10.1287/mnsc.2022.4428. The source file is `refs/replication_package_MS-FIN-21-01990/data/Portfolio_ret.xlsx` (one level above the package root `sdim/sdim/`).

---

## Datasets

| R name | Sheet | Rows | Return cols | Total cols | Date range |
|---|---|---|---|---|---|
| `he2023_factors` | Anomaly | 516 | 70 | 71 | 1974-01-01 – 2016-12-01 |
| `he2023_ff48vw` | FF48vw | 528 | 48 | 49 | 1974-01-01 – 2017-12-01 |
| `he2023_ff30vw` | FF30vw | 528 | 30 | 31 | 1974-01-01 – 2017-12-01 |
| `he2023_ff17vw` | FF17vw | 528 | 17 | 18 | 1974-01-01 – 2017-12-01 |
| `he2023_ff48ew` | FF48ew | 528 | 48 | 49 | 1974-01-01 – 2017-12-01 |
| `he2023_dacheng202` | Dacheng202vw | 552 | 202 | 203 | 1972-01-01 – 2017-12-01 |

The `FF5` sheet is intentionally omitted — it covers a different sample period and is not used as input to any estimator in the paper.

---

## Format

Each dataset is a `data.frame` with:

- **`date`**: class `Date`, first day of each month via `as.Date(paste0(ym, "01"), "%Y%m%d")`.
- **Return columns**: numeric, in percent as supplied by the source (not divided by 100).

---

## Column Naming

All five non-Dacheng sheets (Anomaly, FF48vw, FF30vw, FF17vw, FF48ew) share an identical two-row header structure: Excel row 1 contains `NA` plus numeric column indices (1, 2, 3, …); Excel row 2 contains the actual labels. `skip = 1` skips the index row for all of them, so `readxl` uses the label row as column headers.

- **`he2023_factors`**: names from Anomaly row 2 of the raw sheet (row 1 after `skip = 1`) — e.g., `MKT`, `SMB`, `HML`, …. Five column names contain backslash characters (e.g., `F\_g7`); strip all `\` via `gsub("\\\\", "", nm)` before assigning. The cleaned names are `F_g7`, `Oca_ia`, `Bm_ia`, `Cfp_ia`, `Pchsale_Pchinvt`.
- **`he2023_ff48vw`, `ff30vw`, `ff17vw`, `ff48ew`**: industry names from row 2 of each sheet after `skip = 1` (e.g., `Agric`, `Food`, …). All are already valid R identifiers; use `check.names = FALSE` throughout for consistency.
- **`he2023_dacheng202`**: sequential `p001`–`p202`. The sheet has a two-row hierarchical header (group label row + within-group index row) not suitable for direct column names; use `col_names = FALSE` and skip rows 1–2 manually.

## Processing script (`data-raw/he2023.R`)

Run this script from inside the package directory `sdim/sdim/` with `source("data-raw/he2023.R")`. It reads the Excel file from `../refs/...` (one level up) and writes compressed `.rda` files into `data/`.

```r
# data-raw/he2023.R
# Generates he2023_* datasets from the He et al. (2023) replication package.
# Run from the package root (sdim/) with source("data-raw/he2023.R").
# Requires: readxl

if (!requireNamespace("readxl", quietly = TRUE))
  stop("Package 'readxl' is needed to run this script. Install it with install.packages('readxl').")

xl <- "../refs/replication_package_MS-FIN-21-01990/data/Portfolio_ret.xlsx"

stopifnot(file.exists(xl))

# Helper: yyyymm integer -> Date (first of month)
ym_to_date <- function(x) as.Date(paste0(as.integer(x), "01"), "%Y%m%d")

# ---- he2023_factors (Anomaly sheet) ---------------------------------------
# All five non-Dacheng sheets share an identical two-row header:
#   Excel row 1: NA + numeric column indices (1, 2, 3, ...)
#   Excel row 2: actual labels (MKT, SMB, HML, ... / Agric, Food, ...)
# skip = 1 skips the index row; readxl uses row 2 as column headers.
raw <- readxl::read_excel(xl, sheet = "Anomaly", skip = 1)
nms <- gsub("\\\\", "", colnames(raw))   # strip backslash from 5 factor names
colnames(raw) <- nms
he2023_factors <- data.frame(
  date = ym_to_date(raw[[1]]),
  lapply(raw[-1], as.numeric),
  check.names = FALSE
)

# ---- FF industry portfolios -----------------------------------------------
# FF5 sheet intentionally omitted (different sample period, not used as estimator input).
for (sh in c("FF48vw", "FF30vw", "FF17vw", "FF48ew")) {
  raw          <- readxl::read_excel(xl, sheet = sh, skip = 1)
  colnames(raw)[1] <- "ym"
  df <- data.frame(
    date = ym_to_date(raw$ym),
    lapply(raw[-1], as.numeric),
    check.names = FALSE
  )
  assign(paste0("he2023_", tolower(sh)), df)
}

# ---- he2023_dacheng202 ----------------------------------------------------
# Sheet has two header rows: row 1 = group labels (sparse), row 2 = within-group indices.
raw       <- readxl::read_excel(xl, sheet = "Dacheng202vw", col_names = FALSE)
data_rows <- raw[-(1:2), ]
ym_col    <- as.integer(data_rows[[1]])
ret_cols  <- lapply(data_rows[-1], as.numeric)
names(ret_cols) <- sprintf("p%03d", seq_along(ret_cols))
he2023_dacheng202 <- data.frame(
  date = ym_to_date(ym_col),
  ret_cols,
  check.names = FALSE
)

# ---- Save all datasets ----------------------------------------------------
if (!dir.exists("data")) dir.create("data")
datasets <- c("he2023_factors", "he2023_ff48vw", "he2023_ff30vw",
              "he2023_ff17vw", "he2023_ff48ew", "he2023_dacheng202")
for (nm in datasets) {
  save(list = nm, file = paste0("data/", nm, ".rda"),
       envir = environment(), compress = "xz")
}
message("Done. Run devtools::document() to update NAMESPACE if needed.")
```

## DESCRIPTION changes

Two modifications:

1. `LazyData: false` → `LazyData: true` so datasets are available immediately after `library(sdim)` without requiring `data()` calls.
2. Add `readxl` to `Suggests` (build-time only dependency for the `data-raw/` script).

Add `^data-raw$` so the `data-raw/` directory (containing the processing script) is excluded from the installed package tarball.

## Documentation (`R/data.R`)

One roxygen block per dataset. Each block must include:
- `@title`
- `@description` — what the portfolio set is, that it is from the He et al. (2023) replication package
- `@format` — `A data.frame with N rows and M variables: \describe{\item{date}{...}\item{MKT}{...}...}`
- `@source` — cite He, Huang, Li, Zhou (2023); note this is the published replication package
- `@references` — `\doi{10.1287/mnsc.2022.4428}`
- `@keywords datasets`
- `@examples` — a minimal usage example, e.g. `head(he2023_factors)`, so the manual page is not empty
- Quoted dataset name at end: `"he2023_factors"`
